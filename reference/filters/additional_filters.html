Additional Filters

json​

Converts a string, or object, into JSON format.

When using the JSON output in JavaScript, you don't need to wrap it in quotes because the json filter includes them. The json filter also escapes any quotes inside the output.

Input​

  <script>
    console.log({{ rental.photos | json }});
  </script>

convert_stars​

Converts rental rating to be used as star rating.

Input​

  <!-- Rating = 4.8 -->
    <span class='star-rating rating-{{ rating | convert_stars }}'><span class='sr-only'>{{ rating }}/5</span></span>

Output​

  <span class='star-rating rating-100'><span class='sr-only'>4.8/5</span></span>

iso8601​

Sets date or time in iso8601 format.

Input​

  <meta property='og:updated_time' content='{{ rental.updated_at | iso8601 }}'>

Output​

  <meta property='og:updated_time' content='2015-08-16T12:42:55Z'>

date​

Converts a time/date into another format.

Input​

  {{ article.published_at | date: '%a, %b %d, %y' }}

Output​

  Tue, Apr 22, 14

You can also use a named format from the theme locale files.

Input​

  {{ booking.start_at | date: format: 'date_and_time' }}

Output​

  22/04/2014 15:21

Supported format keys: date, date_and_time, time.

url_for_page​

Returns the URL for a page by id.

Input​

  {{ pages | url_for_page: 12 }}

with_search_param​

Filters categories by a search parameter.

Input​

  {% assign filtered = categories | with_search_param: 'destination', destination.id %}

suggested_rentals​

Returns a list of suggested rentals for a destination.

Input​

  {% assign rentals = destination | suggested_rentals: 6 %}
  {% assign rentals = destination | suggested_rentals: 6, '123,456' %}
