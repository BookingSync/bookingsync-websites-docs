Destination details​

GET /api/public/v2/destinations/:id

Example request: https://www.example.com/api/public/v2/destinations/2924.json?locale=en

Response file: /api_calls/public/destinations/show-response.json

Response body
{
  "data": {
    "id": 2924,
    "title": "Italy",
    "level": 2,
    "parent_id": 101
  }
}
