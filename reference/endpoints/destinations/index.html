Destinations

  1. List destinations
  2. Get a single destination

Parameters

  • OAuth Scopes:
  • public
Name Type Read/Write Description
id Integer Read Destination’s id.
aliases Object Read Destination’s aliases, list of locales described in enums section.
counter String Read Destination’s use counter for rentals.
fullname Object Read Destination’s fullname, list of locales described in enums section.
name Object Read Destination’s name, list of locales described in enums section.
permalink Object Read Destination’s permalink, list of locales described in enums section.

List destinations

List all destinations for given account(s).

GET /destinations
  • OAuth Scopes:
  • public

Response

{
  "destinations": [
    {
      "id": 3212,
      "aliases": {
        "en": null
      },
      "counter": 1,
      "fullname": {
        "en": "Europe, World"
      },
      "name": {
        "en": "Europe"
      },
      "permalink": {
        "en": "world/europe"
      }
    }
  ]
}

Get a single destination

Returns a single destination identified by ID.

GET /destinations/:destination_id
  • OAuth Scopes:
  • public

Response

{
  "destinations": [
    {
      "id": 3212,
      "aliases": {
        "en": null
      },
      "counter": 1,
      "fullname": {
        "en": "Europe, World"
      },
      "name": {
        "en": "Europe"
      },
      "permalink": {
        "en": "world/europe"
      }
    }
  ]
}
