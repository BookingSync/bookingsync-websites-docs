List sources

List all sources for given account.

GET /sources
  • OAuth Scopes:
  • bookings_read / bookings_write

Response

{
  "links": {
    "sources.account": "https://www.bookingsync.com/api/v3/accounts/{sources.account}",
    "sources.bookings": "https://www.bookingsync.com/api/v3/bookings/{sources.bookings}"
  },
  "sources": [
    {
      "id": 1,
      "name": "HomeAway.com",
      "position": 1,
      "created_at": "2015-07-21T10:44:58Z",
      "updated_at": "2015-07-21T10:44:58Z",
      "links": {
        "account": 1,
        "bookings": [
          1,
          3
        ]
      }
    }
  ]
}

Create a new source

Returns a newly created source.

POST /sources
  • OAuth Scopes:
  • bookings_write

Example JSON request

{
  "sources": [
    {
      "name": "HomeAway.com",
      "position": 1
    }
  ]
}

    "sources.account": "https://www.bookingsync.com/api/v3/accounts/{sources.account}",
