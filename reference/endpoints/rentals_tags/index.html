Rentals tags

  1. List rentals tags
  2. Get a single rentals tag

Parameters

  • OAuth Scopes:
  • rentals_read
Name Type Read/Write Description
id Integer Read Rentals Tag’s id.
name Object Read Rentals Tag’s title, list of locales described in enums section.
logo String Read Rentals Tag’s logo (Font Awesome class).
color String Read Rentals Tag’s color (hex).
created_at Time Read Rentals Tag’s create time.
updated_at Time Read Rentals Tag’s update time.

List rentals tags

List all rentals tags for given account.

GET /rentals_tags
  • OAuth Scopes:
  • public

Response

{
  "links": {
    "rentals_tags.account": "http://test.host/api/v3/accounts/{rentals_tags.account}"
  },
  "rentals_tags": [
    {
      "id": 1,
      "color": "#FFFFFF",
      "logo": "fa-rocket",
      "name": {
        "en": "tag"
      },
      "created_at": "2015-07-21T10:44:58Z",
      "updated_at": "2015-07-21T10:44:58Z",
      "links": {
        "account": 1
      }
    }
  ]
}

Get a single rentals tag

  • public
